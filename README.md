# REACT MATERIAL
Material Design Components for React

## Supported Components
* AppBar 
* Buttons 
* Card 
* Checkbox 
* Dialog 
* Drawer Permanent and Temporary)
* Grid (Responsive layout)
* Lists
* Menus
* Progress
* RadioButton 
* Select 
* Simple Menu 
* Slider
* Snackbar
* Switch 
* Tabs 
* Textfield
* Theme 
* Toolbar
* Typography
